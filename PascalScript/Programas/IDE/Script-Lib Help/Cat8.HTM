Rotinas de tratamento de arquivos
Passagem de Parâmetros:
     AZUL = por Cópia
     VERMELHO = por Referência
function DeleteFile ( String ) : Boolean

DeleteFile deletes the file given by FileName


function RenameFile ( String , String ) : Boolean

RenameFile renames the file given by OldName to the name given by NewName


function ChangeFileExt ( String , String ) : String

ChangeFileExt changes the extension of a filename


function FileSearch ( String , String ) : String

FileSearch searches for the file given by Name in the list of
directories given by DirList


function DiskFree ( Integer ) : Integer

DiskFree returns the number of free bytes on the specified drive number
where 0 = Current, 1 = A, 2 = B, etc. DiskFree returns -1 if the drive
number is invalid


function DiskSize ( Integer ) : Integer

DiskSize returns the size in bytes of the specified drive number


function GetCurrentDir : String

GetCurrentDir returns the current directory.


function SetCurrentDir ( String ) : Boolean

SetCurrentDir sets the current directory


function CreateDir ( String ) : Boolean

CreateDir creates a new directory


function RemoveDir ( String ) : Boolean

RemoveDir deletes an existing empty directory
