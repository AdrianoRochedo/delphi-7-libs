TPascalScript 1.7

TPascalScript é uma de um conjunto de classes que encapsulam as características de uma linguagem de Script baseada no Pascal.

Ele inclui um parser, um compilador, um gerenciador de p-code, uma máquina virtual para execução do código e outras classes auxiliares.

Com este pacote você poderá facilmente:

  • Criar Scripts para Controlar Processos
  • Criar Scrips de inicialização de dispositivos
  • Criar Scrips de Instalação
  • Criar uma Linguagem de Macros para uso específico
  • Permitir que usuários personalizem processos e algorítimos
  • Permitir que usuários possam escrever "Plugins"
  • E muito mais ...

    • Principais Características

  • Totalmente orientado a objetos
  • Bloco de definição de variáveis
  • Aceita os tipos: Boolean, Integer, Real, String e Object
  • Acesso a todos os comandos básicos tais como:
    - Comando de atribuição
    - Begin-End
    - Laços While e For
    - If-Then-Else
    - Operadores +, - , *, / div, mod, and, or, not
  • Permite o uso de objetos
  • Permite o acesso a funções e procedimentos externos
  • Compilação com múltiplas mensagens de erro
  • Não contém dependências de plataforma
  • Código fonte claro e altamente expansível

    • Mecanismo

    O mecanismo básico é constituído pelas classes:

  • TPascalScript
    Parser e Compilador

  • TFunctionList (Funções e procedimentos)
    Bibliotecas básicas e do usuário

  • TClassList (Classes que podem ser criadas ou utilizadas no Script)
    Biblioteca de Classes
