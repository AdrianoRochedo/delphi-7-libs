   TeeChart for Piparti 
=============================================

The included example project shows how to preview and print
a Chart component inside a Piparti report.

Two charts are displayed. One in BITMAP format and the other
in METAFILE format.

Please read TeeChart help file regarding metafiles in
"TeeCreateMetafile" function help topic.
