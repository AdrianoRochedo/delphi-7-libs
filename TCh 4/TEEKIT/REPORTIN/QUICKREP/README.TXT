   QuickReport 2.0 Support
=============================================

TeeChart works with QuickReport 2.0 in Delphi 1,
2, 3 and C++Builder 1.0

QuickReport 2.0 can be obtained from QuSoft AS is Norway.
Internet address:  www.qusoft.no

Please refer to the installation instructions
of TeeChart-Pro 3.0 in INSTALL.TXT file to install the
TQRChart component.

There is a demo of QuickReport 2.0 and TeeChart 3.0
in the \Demos\QReport2 folder and here in this folder.

The demo included here, QRDemo.dpr, shows how to print
and preview a TChart component that resides in the same
Form or in another Form. 
Basically it "duplicates" the Chart into the QuickReport
component.


The TQRChart component is implemented in QRTEE.PAS unit.
The QRTEEREG.PAS unit installs and registers TQRChart component.
